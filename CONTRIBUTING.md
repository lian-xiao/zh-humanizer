# Contributing to zh-humanizer

感谢你愿意贡献这个技能。

## 贡献范围

- 新增或修订 AI 写作模式识别规则
- 提供高质量的优化前后对照示例
- 修正文档中的歧义、错别字与不一致约束

## 基本原则

- 不新增原文未提供的事实
- 不引入虚构数据和引用
- 不改变原文立场
- 维持原文风格（尤其学术文本）

## 提交流程

1. Fork 本仓库并新建分支
2. 提交修改并附说明
3. 发起 Pull Request，说明变更目的和示例

## Pull Request 建议

- 小步提交，单个 PR 聚焦一个主题
- 同时更新相关文档（如 `SKILL.md`、`references/ai-patterns.md`）
- 示例尽量覆盖不同文体（学术、商务、通用）
